### Интересные моменты:
- [PeacockTeam](https://github.com/PeacockTeam) - какой-то гид работодателя
- решения задач должны производиться через методы
- глобальные переменные лучше переносить в класс, чтобы в дальнейшем можно было воспользоваться этим классом из другой программы
- Logger сначала нужно инициализировать
- Нужно использовать как можно меньше механизмов упаковки/распаковки, так как это сказывается на производительности
- по возможности лучше избегать операторов break, continue - они приравниваются оператору goto, который нежалательно применять
- LinkedList (связанный список) быстрая, удобная запись и удаление элементов в любом месте списка, но долгое чтение данных из списка, потому что нужно проходиться по всей цепочки элементов до нужной позиции, подходит не для больших списков
- ArrayList (обычный список) быстрое чтение данных, но медленее запись и удаление.
- [mvnrepository](https://mvnrepository.com/) - одно из самых крупных хранилищ библиотек java
- Чтобы добавить библиотеку к проекту, нужно перейти в Java Projects => Referenced Libraries => Add Librari +
