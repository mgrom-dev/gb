![Задача 3](task3.jpg)
```
numbers = [2, 5, 13, 7, 6, 4]
size = 6
index = 0
max_idx = 0
max = numbers[max_idx]
while (index < size)
    if (numbers[index] > max) then
        max_idx = index
        max = numbers[index]
    index = index + 1
numbers[max_idx] = numbers[size - 1]
numbers[size - 1] = max
print(numbers)
```
